Wireless Text-to-Cloud Leaderboard

This project integrates an embedded IR remote texting interface with an AWS cloud-hosted leaderboard. Players use a CC3200 LaunchPad to enter messages using a multi-tap IR remote system, which sends results wirelessly to an AWS Lambda function. The function updates a DynamoDB table, and the latest scores are retrieved and displayed here in real time.
